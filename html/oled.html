Control OLED Screen - DigitalPlat Dashboard

1.The OLED screen must be initialized/cleared before setting the text

Initialize OLED screen / clear content

2.Here you can set the display typesetting to display multi-line text. For example, you can set your first row of text, and then set the X coordinate to 0 and the Y coordinate to 0, which will set your first row of characters starting from the first character in the first row. Return here after execution, and then set the text displayed in the second row again, and set the X coordinate to 5 and the Y coordinate to 20. After execution, the corresponding text will be set in the second row, and so on. After the setting is complete, click Show below to print out the text layout you set. Isn't it interesting?

3.Well, the last step, click the Show button below to display the text layout you set!
